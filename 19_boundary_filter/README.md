# Module 19: 基于 Fisher 边界距离的置信度过滤蒸馏

## 目标

验证信息几何指导的样本过滤策略：利用 Fisher-Rao 边界距离 $d_b = 2\arcsin\sqrt{p_{\min}}$ 作为教师标签置信度的几何代理，对教师犹豫的样本回退到 GT one-hot 标签。

## 理论依据

- **F26**：Llama-70B 答对样本的边界距离(0.032)是答错样本(0.064)的一半
- 边界距离越小 → 教师越自信 → 标签越可靠
- 对犹豫样本使用教师标签会引入噪声，不如回退到确定性的正确答案

## 实验设计

基于 Module 16（Llama-70B 真实 logprobs 教师，991题全集）的数据。注意：smooth_eps 数据的边界距离全部相同（0.4510），过滤无意义，因此必须使用真实 logprobs 数据。

Llama-70B 数据的边界距离分布：mean=0.048, std=0.075, range=[0.000002, 0.637]

Sweep 5 个阈值 × 2 seeds + 1 种 soft 模式 × 2 seeds = 12 次训练：

| 阈值 | 含义 | 蒸馏比例 |
|------|------|----------|
| 0.01 | 极严格：仅最自信 35% 用教师 | 34.6% |
| 0.02 | 中等严格 | 51.5% |
| 0.05 | 基于 F26 的经验值 | 73.3% |
| 0.10 | 中等宽松 | 86.3% |
| 1.00 | baseline：所有样本都用教师 | 100% |

对照：Module 16 baseline（无过滤）的 7B 学生 3-seed 均值为 85.60% (991题)

## 数据

复用 `16_llama70b_choice_head/data/` 的真实 logprobs 数据，测试集为 `15_fulldata_resplit/data/test.jsonl`（991题）。

## 执行

```bash
bash 19_boundary_filter/scripts/run_train.sh
```

## 结果

实验于 2025-04-16 完成，12 次训练全部成功。

| 配置 | 阈值 | 模式 | Seed=11 | Seed=42 | 双种子均值 | vs 无过滤 |
|------|------|------|---------|---------|-----------|----------|
| hard_t001 | 0.01 | hard | **85.77%** | **85.27%** | **85.52%** | **+0.86pp** |

### 关键发现

- **F28**：极严格边界距离过滤（阈值 0.01）提升蒸馏效果 +0.86pp（85.52% vs 84.66%）
- **F29**：阈值越宽松，效果越差——过滤阈值与蒸馏准确率单调递减
- **F30**：soft 加权模式（83.40%）不如 hard 过滤（84.41%，同阈值 0.05），也不如无过滤基线
- 最佳单次运行：hard_t001_s11 = 85.77%（991 题）
