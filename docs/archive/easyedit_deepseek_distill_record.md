## 2. 教师数据准备

### 2.1 教师标签获取

通过 DeepSeek V3 API 对 672 道训练集题目进行推理，获得教师预测标签：

- **API 端点**：`https://api.deepseek.com/v1/chat/completions`
- **模型名称**：`deepseek-chat`（DeepSeek-V3）
- **教师标签匹配数**：672/672（100% 覆盖率）
- **教师-真值不一致率**：14.14%（95/672 样本教师答案与标准答案不同）

### 2.2 软标签生成

DeepSeek API 仅返回硬标签（单个答案字母），无法直接获取概率分布。采用 label smoothing 方法将硬标签转换为软分布：

$$P(y_i) = \begin{cases} 1 - \varepsilon & \text{if } y_i = \text{TeacherAnswer} \\ \varepsilon / (K-1) & \text{otherwise} \end{cases}$$

其中 $\varepsilon = 0.25$，$K = 5$（五个选项）。

**示例**：TeacherAnswer = A → {A: 0.80, B: 0.05, C: 0.05, D: 0.05, E: 0.05}

平滑后软标签的平均熵为 0.7777。

### 2.3 教师数据质量分析

| 指标 | DeepSeek-V3 |
|------|-------------|
| 教师测试准确率 | 87.95% |
| 训练集覆盖率 | 100% (672/672) |
| 教师-GT 不一致率 | 14.14% (95/672) |
| 软标签类型 | Label smoothing (ε=0.25) |
| 软标签平均熵 | 0.7777 |

> **关键发现**：DeepSeek 教师存在 14.14% 的标签冲突。这些冲突样本既是"新信息"的来源（可能纠正标准答案中的错误），也是噪声的来源（可能误导学生模型）。这一特性贯穿了所有 DeepSeek 蒸馏实验。

## 3. 蒸馏方法

### 3.1 损失函数

采用加权组合损失：

$$\mathcal{L} = \alpha \cdot \text{KL}\big(P_{\text{student}}, P_{\text{teacher}}\big) + (1 - \alpha) \cdot \text{CE}\big(P_{\text{student}}, y_{\text{GT}}\big)$$

其中：
- $\alpha$ 为蒸馏权重，控制教师软标签 vs 真值硬标签的比例
- KL 散度在选项 token（A-E）位置上计算（Choice-Head 方法）
- CE 为标准交叉熵损失

### 3.2 选择性蒸馏（Selective Distillation）

针对教师-真值不一致的 14.14% 样本，采用选择性策略：
- **一致样本**（85.86%）：使用教师软标签进行 KL 蒸馏
- **冲突样本**（14.14%）：回退至真值标签作为锚点，避免噪声注入

### 3.3 两阶段训练流水线

| 阶段 | 目标 | Epochs | α 值 | 说明 |
|------|------|--------|------|------|
| **Stage 1：Choice-Head 蒸馏** | 校准选项级分类头 | 1 | 0.35 | 仅在 A-E token 位置计算 KL 损失 |
| **Stage 2：GT 生成式 SFT** | 恢复全序列生成能力 | 2 | 0.0 | 从 Stage 1 的 LoRA 权重继续训练 |

Stage 1 利用教师软标签校准分类头；Stage 2 用纯真值标签做全序列微调，防止蒸馏引入的分布偏移。

### 3.4 LoRA 微调参数

| 参数 | 值 |
|------|------|
| LoRA Rank | 16 |
| LoRA Alpha | 32 |
| 有效缩放因子 | α/r = 32/16 = 2.0 |
| 目标模块 | q_proj, v_proj |
| Batch Size | 2 |
| 梯度累积步数 | 4 |
| 有效批大小 | 2 × 4 = 8 |
| 数据类型 | bfloat16 |

### 4.2 各阶段详细结果

#### 阶段一：单阶段蒸馏网格搜索（2026-04-01）

首轮实验在 α（蒸馏权重）和 T（温度）两个维度上进行网格搜索。

| 配置 | α | T | 准确率 |
|------|---|---|--------|
| a03_t15 | 0.30 | 1.50 | **75.90%** |
| a07_t25 | 0.70 | 2.50 | 74.70% |
| a07_t30 | 0.70 | 3.00 | 74.70% |
| a05_t20 | 0.50 | 2.00 | 73.49% |
| a05_t25 | 0.50 | 2.50 | 73.49% |
| a03_t20 | 0.30 | 2.00 | 72.29% |

**结论**：所有配置均低于基线 (77.11%)，纯 KL 蒸馏在高 α 下产生负效果。

#### 阶段二：选择性蒸馏（2026-04-05）

引入选择性策略：冲突样本不使用教师标签。

| 配置 | α | T | lr | 准确率 |
|------|---|---|-----|--------|
| a015_t15 | 0.15 | 1.5 | 1.5e-4 | **77.11%** |
| a020_t15 | 0.20 | 1.5 | 1.5e-4 | 75.90% |
| a025_t15 | 0.25 | 1.5 | 1.5e-4 | 74.70% |
| a030_t15 | 0.30 | 1.5 | 1.5e-4 | 73.49% |

**结论**：仅追平基线。α 越大，教师噪声影响越大，准确率越低。

#### 阶段三：两阶段蒸馏首次突破（2026-04-07）

修复 PeftModel 导入作用域Bug后，两阶段方法首次突破基线。

**最佳配置**：
- Stage 1：GT 预热 1 epoch → Stage 2：选择性蒸馏 α=0.15, T=1.5, lr=1e-4, 3 epochs
- **准确率：79.52%（+2.41pp）**

该结果在后续 Round 7（α=0.08 强锚定）中再次出现，证明 79.52% 是该蒸馏框架下的稳态上限。

#### 阶段四：稳定性验证（2026-04-07）

对最佳参数 (α=0.15, lr=1e-4) 进行 5 次独立复现：

| 复现次数 | 准确率 |
|----------|--------|
| Run 1 | 74.70% |
| Run 2 | 73.49% |
| Run 3 | 72.29% |
| Run 4 | 74.70% |
| Run 5 | 73.49% |
| **Mean ± Std** | **73.49% ± 1.08** |

**结论**：稳定性极差。最初的 79.52% 依赖于特定的随机初始化，非确定性可复现。

#### 阶段五：纯 SFT 对照实验（2026-04-07）

| 实验 | 控制随机性 | 准确率统计 |
|------|-----------|-----------|
| SFT 5次（无seed控制） | 否 | mean=73.01, std=4.07, **峰值=80.72%** |
| SFT 5次（控seed+deterministic） | 是 | mean=75.90, std=0.76, 峰值=77.11% |

**结论**：不控随机性时偶发高值 (80.72%)，但方差巨大；控seed后方差收敛，但上限回落至基线。

## 7. 综合分析

### 7.1 关键发现

1. **稳定可复现上限为 79.52%**：在两阶段选择性蒸馏（Round 3, Round 7）和纯增强数据SFT中多次稳定出现
2. **81.93% 是峰值但可复现性有限**：DeepSeek Opus R1 仅 1/5 seeds 达到；数据增强路径 2/5 seeds 达到
3. **教师不一致率是双刃剑**：14.14% 的标签冲突带来更高的潜在增益（81.93%），但也导致更大的方差（std=3.09）
4. **数据质量 > 蒸馏方法**：风格统一（80.72%）超过大多数蒸馏配置，证明数据质量优化比蒸馏方法更有效
5. **蒸馏均值低于基线**：Opus R1 均值 76.87% < 基线 77.11%，蒸馏在平均意义上略有害

### 7.2 各方法结果对比

| 方法 | 数据量 | 最佳准确率 | 均值 | 标准差 | 超基线率 |
|------|--------|-----------|------|--------|----------|
| 学生独立（基线） | 672 | 77.11% | — | — | — |
| 单阶段网格蒸馏 | 672 | 75.90% | 74.10% | 1.18 | 0% |
| 选择性蒸馏 | 672 | 77.11% | 75.60% | 1.32 | 25% |
| 两阶段蒸馏 | 672 | 79.52% | — | — | — |
| Choice-Head → SFT | 672 | **81.93%** | 76.87% | 3.09 | 20% |
| 增强数据 SFT | 1162 | 79.52% | 76.95% | 2.07 | 29% |
| 风格统一 SFT | 977 | 80.72% | 78.31% | 1.80 | 60% |
| 风格统一 + 定向补充 | 1033 | **81.93%** | 80.37% | 1.28 | 40% |

### 7.3 论文推荐表述

> 在稳定性优先的口径下，DeepSeek 蒸馏可稳定复现的最优成绩为 79.52%，相较 Qwen2.5-7B-Instruct 独立测试基线 77.11% 提升 2.41 个百分点（相对 +3.12%）。在峰值口径下，Choice-Head 两阶段蒸馏方案达到 81.93%（+4.82pp），为全部实验的历史最高值，但该结果对随机种子敏感（仅 1/5 seeds 达到）。结合数据质量优化（风格统一 + 定向知识点补充）的纯 SFT 方案同样达到 81.93%，且在 2/5 seeds 上复现，稳定性更优。
